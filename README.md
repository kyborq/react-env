# React Environment

For developing React app.

## Install and using

Installing

```
npm install
```

How to use for webpack dev-server

```
npm start
```

Building

```
npm build
```

## Structure

Files and folder structure for example:

```
src
| components
  | Component.jsx
  | Component.css
| pages
  | Page.jsx
  | Page.css
| App.css
| App.js
| index.html
| index.js
```
